Al-Sultan Loyalty System

v1.0.0

This is a mini loyalty system that maintains loyalty points of clients in Al-Sultan supermarket & Stores

Usage Guide
  • Add new user in File > Add User with barcode, name and other optional fields
  • Open the settings in File > Settings and setup amount per loyalty point
  • In the Home menu (File > Home), scan user barcode and enter amount purchased in store to add Loyalty points. The calculation is (amount purchased)/(amount per loyalty point).
  • To redeem loyalty points go to File > Redeem points and scan user bardcode. Then enter the amount to redeem which must be less than the amount of points the user has.
